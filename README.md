## React脚手架
### 目的

1. 快速搭建react项目
2. 完善的编译配置

### 使用方法

1. npm i
2. npm start

### 内容

1. webpack
2. webpack-dev-server
3. react
4. react-router
5. eslint
6. 其他功能请参考package与webpack.config.js

### 代码样例

1. App: 图片引用，less编译
2. Render: 根据类型与组件类绘制组件
3. Context: react context跨组件传输数据
4. Control: 受控组件阻止默认行为
5. Hoc: 高阶组件，组件包装
